Problema 1: Ficha Digital de Empleado

Enunciado del Problema

Escribe un programa en C# que solicite al usuario los datos básicos de un empleado: nombre completo, edad, cargo, salario bruto y si cuenta con seguro médico (true/false). Al finalizar la captura, debe mostrar una ficha formateada con los datos ingresados.

Conceptos a Utilizar
  • Variables: string, int, double, bool
  • Entrada de texto: Console.ReadLine()
  • Conversión de tipos: Convert.ToDouble(), Convert.ToBoolean()
Ejemplo de Salida Esperada 
--- FICHA DE EMPLEADO ---
Empleado: Roberto Gómez
Cargo: Analista de Sistemas
Edad: 28 años
Salario: $850.50
Seguro Activo: True

Problema 2: Calculadora de Promedio de 3 Notas

Enunciado del Problema

Crea un programa que pida el nombre de una asignatura y las notas de 3 Evaluaciones Continuas (Laboratorios). El programa debe calcular el promedio final sumando las tres notas y dividiéndolas entre 3, imprimiendo el resultado exacto con sus decimales.

Conceptos a Utilizar
  • Tipo de dato decimal: double
  • Operaciones aritméticas simples (+, /)
  • Uso de paréntesis para prioridad de operaciones.
Ejemplo de Salida Esperada 
Materia: Programación Estructurada
Ingrese Nota 1: 8.5
Ingrese Nota 2: 9.0
Ingrese Nota 3: 7.5

Promedio Final: 8.33

Problema 3: Ticket de Compra Rápida (con IVA)

Enunciado del Problema

Desarrolla una aplicación de consola que simule el cobro en una tienda. Debe solicitar el nombre del producto, el precio unitario y la cantidad comprada. Con una constante de IVA del 13% (0.13), calcula el Subtotal, el valor del Impuesto y el Total Final a pagar.

Conceptos a Utilizar
  • Declaración de constante: const double IVA = 0.13;
  • Multiplicación de variables numéricas.
  • Formateo claro de datos de salida numéricos.
Ejemplo de Salida Esperada 
Producto: Teclado Mecánico
Cantidad: 2 | Precio c/u: $45.00
--------------------------------
Subtotal: $90.00
IVA (13%): $11.70
Total A Pagar: $101.70

Problema 4: Conversor de Unidades de Longitud

Enunciado del Problema

El usuario ingresará una distancia dada en Metros. El programa utilizará constantes para convertir y mostrar esa misma distancia en: Centímetros, Milímetros y Pulgadas (1 metro = 39.3701 pulgadas).

Conceptos a Utilizar
  • Uso de constantes numéricas de conversión.
  • Reutilización de una misma variable de entrada para múltiples cálculos.
  • Estructuración organizada de texto de salida.
Ejemplo de Salida Esperada 
Ingrese la distancia en metros: 2.5

--- CONVERSIONES ---
Centímetros: 250 cm
Milímetros:  2500 mm
Pulgadas:    98.4251 in

Problema 5: Control de Versiones con Git

Enunciado del Problema

Crea un repositorio local llamado SolucionEjercicios01. Agrega el proyecto en C# desarrollado en los ejercicios anteriores y realiza el flujo de Git necesario para publicar el proyecto en tu cuenta de GitHub.

Secuencia de Pasos Requerida en la Terminal 
# 1. Inicializar repositorio
git init

# 2. Agregar todos los archivos creados
git add .

# 3. Crear el primer commit oficial
git commit -m "feat: Agregar ejercicios practicos de la Sesion 01"

# 4. Vincular y subir al repositorio remoto de GitHub
git remote add origin https://github.com/tu_usuario/SolucionEjercicios01.git
git push -u origin main
